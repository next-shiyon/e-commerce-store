# React + Typescript 학습용 EC 사이트 개발

## 어플리케이션 실행 방법 📄

```shell
# 프로젝트 클론
git clone git@github.com:next-shiyon/e-commerce-store.git
cd e-commerce-store

# 모드 모듈 설치
yarn
# 어플리케이션 실행
yarn run dev
```

## URL 설계 🏛️

| URL                  | 페이지                   |
| -------------------- | ------------------------ |
| /                    | 메인             |
| /products            | 상품리스트         |
| /products/:productId | 상품상세           |
| /products/cart       | 쇼핑카트 |
| /products/register   | 상품등록           |
